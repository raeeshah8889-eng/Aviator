# Quick Start Guide

## Prerequisites
- Node.js 16.x or higher
- npm or yarn

## Setup & Run

### 1. Install Dependencies

```bash
# Server dependencies
cd server
npm install
cd ..

# Client dependencies
npm install --legacy-peer-deps
```

### 2. Start the Backend (Terminal 1)

```bash
cd server
npm start
```

The server will run on http://localhost:5000

### 3. Start the Frontend (Terminal 2)

```bash
npm start
```

The React app will run on http://localhost:3000

## Game Features

✅ Plane flying animation with real-time multiplier
✅ $10,000 starting wallet balance
✅ 100-120 AI bots playing each round
✅ Place bets during countdown phase
✅ Cash out before crash to win
✅ Auto-cashout feature
✅ Live players panel with all participants
✅ Leaderboard showing top winners
✅ Deposit funds functionality
✅ Round history display
✅ Provably fair crash points

## How to Play

1. Wait for the 5-second countdown
2. Enter your bet amount (minimum $1)
3. Optionally set auto-cashout multiplier
4. Click "Place Bet" during countdown
5. Watch the plane fly and multiplier grow
6. Click "Cash Out" before the crash!

Good luck! 🚀✈️
